# 研究笔记目录

本目录用于存放项目相关的研究笔记、思考记录和讨论草稿。

## 目录结构

```
research-notes/
├─ 2025-05/                 # 按月份组织
│  ├─ water-cycle-modes.md  # 水文模式分类思考
│  └─ region-selection.md   # 流域选择策略
└─ README.md
```

## 笔记类型

- **技术笔记**：架构决策、技术选型
- **研究笔记**：领域知识、方法论
- **会议记录**：讨论和决策
- **读书笔记**：文献阅读摘要

## 格式建议

```markdown
# 标题

日期: YYYY-MM-DD
标签: #标签1 #标签2

## 背景

...

## 内容

...

## 下一步

...
```

## 注意事项

- 研究笔记是个人/团队内部文档，不作为正式项目文档
- 定期整理有价值的内容迁移到正式文档
- 保持简洁，避免过度格式化
